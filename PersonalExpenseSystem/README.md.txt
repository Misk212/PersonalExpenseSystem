Sorgente_Python_Gestione_Spese

Interprete utilizzato: Python 3.14.2; L'esecuzione del programma non richiede compilatore in quanto Python esegue lo script come interprete.

Librerie aggiuntive richieste:

psycopg2 per gestione DB tramite interprete Python
datetime carica le librerie standard per la gestione degli intervalli temporali.

Istruzioni di avvio del programma:

Assicurarsi di avere Python 3.14.2 installato nel sistema.
Assicurarsi di avere PostgreSQL in esecuzione.
Creare il database eseguendo lo script SQL presente nella cartella sql/.
Configurare correttamente i parametri di connessione al database nel file Python.
Posizionarsi nella cartella contenente il file main.py.
